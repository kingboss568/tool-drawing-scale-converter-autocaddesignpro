This template turns contrast checker into a repeatable structure. Use it as a starting point, then adjust fields for the actual project, client, standard, or pricing model.

Recommended sections: objective, inputs, assumptions, result, review owner, open risks, and export note. The template is intentionally explicit because vague utility pages create weak downstream decisions.

Copy the result into the working document only after verifying the assumptions. Static tools are most useful when they shorten the first draft while leaving final judgment visible.
